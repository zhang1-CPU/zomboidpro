Build 42 Mod Compatibility Guide: Working, Broken & Alternatives

Updated: May 10, 2026 ~8 min read

Build 42 brought massive changes to Project Zomboid's engine, and many popular mods from Build 41 are broken or incompatible. This guide tracks which mods work, which are broken, and provides alternatives for the most popular broken mods.

Important: Mod compatibility changes with each B42 hotfix. This guide was last updated for Build 42.17. Always check the mod's Workshop page for the latest compatibility status before installing.

Multiplayer-Compatible Mods

These mods work in multiplayer sessions (all players must have them installed):

  • Common Sense – All players need it for synchronized actions.
  • Has It Read? – Client-side only, does not affect server.
  • Equipment UI – Client-side only, safe for multiplayer.
  • Better Sorting – Client-side only, does not sync with other players.
  • Simple Overhaul: Weapons & Attachments – Server must have it, clients need it for compatibility.

Multiplayer Warning: If any player joins a server with incompatible mods, they may experience crashes or desync. Always verify all mods are server-compatible before joining a multiplayer session.

How to Check Mod Compatibility

  1. Check the Workshop page. Mod authors usually update the description with compatibility status.
  2. Look for "B42" or "Build 42" tags. Mods updated for B42 will have this tag in their title or description.
  3. Read recent reviews. Other players will report if a mod works or crashes in B42.
  4. Test in a new save first. Never install untested mods on your primary save. Create a new world and test first.
  5. Disable one mod at a time. If you experience crashes, disable mods one by one to find the culprit.

Mod Installation Tips for B42

  • Subscribe via Steam Workshop. The easiest way to install mods. Steam handles updates automatically.
  • Check load order. Some mods must load before or after others. Read the mod description for load order instructions.
  • Keep mods updated. Enable automatic mod updates in Steam settings.
  • Backup your saves. Mods can corrupt saves. Always backup before installing new mods.
  • Use a mod profile. Create different mod profiles for single-player and multiplayer sessions.

B42 Mod Checklist

  • ✅ All mods verified as B42-compatible
  • ✅ Load order configured correctly
  • ✅ Multiplayer mods synced across all players
  • ✅ Save backup created before mod installation
  • ✅ Tested in a new world before using on primary save
  • ✅ Mod profiles set up for different game modes

Recommendation: Start with a minimal mod list (Common Sense + Has It Read? + Equipment UI). Add more mods only after verifying stability. Less is more in Build 42 – the vanilla game is significantly improved over Build 41.
