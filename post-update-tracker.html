Build 42 Update Tracker: All Patches & Changes

New May 9, 2026 Build 42 ~5 min read

Project Zomboid receives frequent patches during the Build 42 development cycle. This page tracks all major updates with player-friendly summaries so you know what changed and how it affects your game.

Note: Build 42 is still in active development. Updates may introduce bugs, break mods, or change game mechanics. Always backup your saves before updating.

Latest Update: Build 42.17 (May 2026)

  • Vehicle system overhaul: New engine simulation with 18 parts and 4 fluids
  • Mod compatibility fixes: Several popular mods now work with B42 format
  • Zombie pathing improvements: Better navigation through buildings
  • Bug fixes: Fixed plumbing, thread consolidation, and smoker trait issues
  • Multiplayer stability: Reduced server crashes and desync issues

Previous Major Updates

Build 42.15 (April 2026)

  • New map interiors: Many buildings now have explorable interiors
  • Farming improvements: New crop types and growth mechanics
  • UI updates: Improved inventory and character panels

Build 42.12 (March 2026)

  • Seasonal weather system: Full spring, summer, autumn, winter cycle
  • Temperature mechanics: Hypothermia and heat exhaustion added
  • Water system changes: Rain collectors and wells updated

Build 42.08 (February 2026)

  • Initial B42 release: Major overhaul from B41
  • Graphics engine update: Improved lighting and shadows
  • New mod format: B41 mods are incompatible

How to Stay Updated

  • Official channels: Follow PZ Wiki and the official Project Zomboid forums
  • Steam announcements: Check the game's Steam page for patch notes
  • Bookmark this page: We update this tracker within 24 hours of every official patch

Before every update: Backup your save files. Build 42 patches can corrupt saves, especially during early development. See our Save Safety Guide for detailed instructions.
