Build 42 Multiplayer Setup Guide: Server, Co-op & Connection Fix

Written for Build 42 2026-05-11 ~6 min read

Build 42 has changed multiplayer significantly. Whether you want to host a dedicated server, play co-op with friends, or fix connection errors, this guide covers everything you need.

How to Create a Multiplayer Server

Method 1: Host from Game Client (Easiest)

  1. Launch Project Zomboid and select Host from the main menu.
  2. Choose Create New Server.
  3. Enter a Server Name and set a Password.
  4. Select your Sandbox Settings.
  5. Click Start Server. Wait for the world to generate.
  6. Your friends can now find your server in the Internet tab.

Method 2: Dedicated Server (Advanced)

  1. Open Steam, right-click Project Zomboid ?Properties ?Installed Files.
  2. Find ProjectZomboidDedicatedServer64.bat.
  3. Run the script and follow the setup wizard.
  4. Forward port 16261 (UDP) on your router.

Common Connection Errors & Fixes

ErrorCauseFix
"Connection Timed Out"Firewall or incorrect IPCheck port forwarding, verify IP
"Mod Mismatch"Different modsEnsure all players have same mods
"Incompatible Version"Game version mismatchAll players must update to same version

Note: Build 42 multiplayer is still being optimized. Expect occasional server crashes and desync.
